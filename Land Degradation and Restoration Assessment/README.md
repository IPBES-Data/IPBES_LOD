This directory contains the IPBES Assessment Report on Land Degradation and Restoration in LOD format.

The structure of the file follows the IPBES ontology version 06:
https://github.com/IPBES-Data/IPBES_Ontology

The assessment report is published in 2018 and consists of 8 chapters and a Summary for Policy Makers (SPM).

For more information about this report, see: https://www.ipbes.net/assessment-reports/ldr 

For the full report, see: https://doi.org/10.5281/zenodo.3237393

For information on how to cite the report, see: https://ict.ipbes.net/ipbes-ict-guide/data-and-knowledge-management/technical-guidelines/suggested-citations#i.-transformative-change-assessment
